The only bot you'll ever need.

A variety of commands. Not.

To make it short, there are enough bots which have a shitton of commands that you'll never really use.
So we cropped our commands to the minimum to annoy you as little as possible.

There is still everything you need - and some commands are grouped into one.
The ultimate Discord tool. In one Application.

The commands are as flexible as possible. They adapt to every server and show you exactly what you need.

No matter if you're a developer or a server administrator, you'll see everything you need at one spot.
We didn't forget about the fun.

After all, the fun didn't come too short for this project. We included a currency system that features daily credits, a shop and also trading with other users.
